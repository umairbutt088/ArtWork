## Features

- Read NFC tags
  - uid
  - NFC technology
  - NDEF
- Write NDEF
  - RTD_URI
    - url, email, sms, tel, or custom scheme
  - RTD_TEXT
  - WIFI SIMPLE RECORD
- Toolkits
  - NfcA
    - Custom transceive
    - Erase all
    - Format to NDEF
  - IsoDep
    - Custom APDU (mostly Android)
- Save your own records
